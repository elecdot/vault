# 1 研究设计

## 正式题目

**专精特新政策对企业创新的因果效应——基于渐进式DID与政策文本挖掘的双层分析**

## 1.2 核心研究问题

**主问题：** 专精特新小巨人的认定是否显著促进企业创新产出（专利数量、研发投入）吗？

**子问题：**
- 不同省份的配套政策工具强度是否影响政策效果？（供给型/需求型/环境型哪类更有效？）
- 不同地区（经济水平、技术聚集程度）的企业获得的政策效果有差异吗？

## 1.3 创新亮点

**因果识别严谨：** 用分批认定作为外生冲击做渐进式DID，回应第一此找老师面谈提及的“到底是政策导致创新还是创新好才被选上”的问题。

**两层分析结合：** 政策内容的文本量化（政策长什么样）、因果效应估计（政策有没有用），两层之间通过异质性分析衔接。

# 2 研究设计 方法核心

## 2.1 两层分析结构

| 层次   | 研究内容                                       | 方法                                  | 产出                     |
| ------ | ---------------------------------------------- | ------------------------------------- | ------------------------ |
| 第一层 | 各省专精特新配套政策文本分类，构建政策强度指数 | 爬虫获取文本 → RoBERTa分类 → 构建指数 | 省份×年份的政策强度面板  |
| 第二层 | 专精特新认定对企业创新的因果效应               | 渐进式DID（多期处理）                 | 因果效应系数、平行趋势图 |
| 连接层 | 政策强度是否调节认定效果                       | 在DID中引入交互项做异质性分析         | 异质性回归表             |

## 2.2 第一层 政策文本分类

### 2.2.1 政策工具分类框架

采用Rothwell & Zegveld（1982）经典三分法，与上届参考论文一致，评委熟悉：

| 政策类型 | 核心特征               | 常见表现形式                           |
| -------- | ---------------------- | -------------------------------------- |
| 供给型   | 政府直接提供资源给企业 | 资金奖励、人才引进、基础设施、研发补贴 |
| 需求型   | 政府创造或引导市场需求 | 政府采购、示范项目、应用推广、外包支持 |
| 环境型   | 优化企业经营的制度环境 | 税收减免、金融担保、法规制定、知识产权 |

### 2.2.2 文本来源与收集范围

收集范围：全国31个省份，2020-2025年发布的省级专精特新配套政策文本

主要来源：各省政府官网、工信部中小企业司（zjtx.miit.gov.cn）

关键词：专精特新、小巨人、中小企业

### 2.2.3 分类方法

基于RoBERTa预训练模型（hfl/chinese-roberta-wwm-ext-large）？做文本分类

**分类流程：** 文本清洗 → 关键词提取（TF-IDF）→ RoBERTa微调分类 → 输出三类标签

产出：各省各年供给型/需求型/环境型政策数量，加权构建政策强度综合指数

## 2.3 第二层：渐进式DID因果识别

### 2.3.1 核心变量定义

| 变量类型 | 定义                             | 数据来源           |
| -------- | -------------------------------- | ------------------ |
| 因变量   | 企业年度发明专利申请量（取对数） | CSMAR专利数据库    |
| 因变量   | 企业研发投入/营业收入（%）       | CSMAR财务数据库？  |
| 处理变量 | 渐进式DID交互项（认定后各年=1）  | 工信部公告名单     |
| 调节变量 | 所在省份政策强度综合指数         | 第一层文本挖掘产出 |
| 控制变量 | 企业规模（总资产对数）           | CSMAR？            |
| 控制变量 | 企业成立年限                     | CSMAR？            |
| 控制变量 | 资产负债率                       | CSMAR？            |
| 控制变量 | 净资产收益率                     | CSMAR？            |

### 2.3.2 样本说明

- **处理组：**工信部六批次国家级专精特新小巨人上市公司，共1044家？（A股上市公司）
	- 第一批：37家，第二批：156家，第三批：227家，第四批：325家，第五批：161家，第六批：137家
- **控制组：**同期未被认定为小巨人的上市公司
- **时间范围：**2015-2025年（保证每家企业认定前至少有3年基期数据）

### 2.3.3 基准回归方程

$$
\ln(patent_{it}) = \alpha + \beta_1 (treat_i \times post_{it}) + \gamma X_{it} + \mu_i + \lambda_t + \varepsilon_{it}  
$$

- $\mu_i$: Firm fixed effects (time-invariant characteristics)
- $\lambda_t$: Year fixed effects (macro shocks)
- $\beta_1$: Key coefficient measuring causal effect
- Standard errors clustered at the firm level

### 2.3.4 检验

| 检验类型     | 目的                   | 具体做法                           | 预期结果                         |
| ------------ | ---------------------- | ---------------------------------- | -------------------------------- |
| 平行趋势检验 | 验证DID前提假设        | 事件研究法，画认定前各年系数图     | 认定前系数不显著，认定后显著上升 |
| 安慰剂检验   | 排除偶然性             | 随机打乱处理时间1000次，看系数分布 | 真实系数在随机分布的右尾         |
| PSM-DID      | 提高处理组控制组可比性 | 先用倾向得分匹配控制组再做DID      | 系数符号方向一致                 |
| 排除预期效应 | 排除企业提前响应       | 去掉认定前一年重新回归             | 结果稳健                         |

## 2.4 异质性分析

1. **按省份政策强度分组：** 将各省按政策强度指数分为高/低两组，比较认定效果差异
2. **按地区经济发展水平分组：** 东部/中西部企业的政策效果是否不同
3. **按政策工具类型分解：** 供给型强度高的省份 vs 需求型强度高的省份，认定效果孰优
