# JY901-I2C

## Focus
How to design AXI-I2C-JY901 IP.

## Notes

### Addressing

JY901 default using 7-bit address:
```
0x50 (0b1010000)
```

Since I2C protocol use a Byte for [[20-i2c#Addressing and R/W Control]]:
```
Write a Byte = 0x50 << 1 | 0 = 0xA0
Read a Byte = 0x50 << 1 | 1 = 0xA1
```
